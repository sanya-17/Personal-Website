ECE 0302 - Data Structures and Algorithms

In this course, we studied and implemented fundamental data structures like stacks, queues, trees, lists, and heaps. We also learned about the analysis of algorithms through topics like big O notation and algorithm growth rates.

ECE 0202 - Embedded Processors and Interfacing

In this course, we were introduced to microprocessors and interface circuitry. We learned the basic structure of ARM microcontrollers, including registers, data memory, instruction memory, GPIO structures, timers, counters, and interrupts.

Furthermore, we were able to interface with embedded systems circuitry like buttons and stepper motors by writing C and assembly code in the Keil IDE.

ECE 0402 - Signals, Systems, and Probability

In this course, we studied several topics regarding signals and systems including analysis of LTI systems, application of Laplace transform to linear circuits and LTI systems, and the Fourier Transform. Furthermore, we used Matlab to visualize the behavior of signals and systems.

We also studied topics regarding probability and statistics including random variables, probability density functions, and hypothesis testing.

ECE 0301 - Problem Solving in C++

In this course, we took a deep dive into C++ and its applications. We focused especially on pointers, memory allocation, and the four principles of object-oriented programming. Some of the assignments from this course included a circuit simulator and differential equation solver.

ECE 0201 - Digital Circuits and Systems

In this course, we received an introduction to digital circuit design. We applied what we learned in class through laboratory sessions where we programmed digital circuits in VHDL on an FPGA. The final laboratory for this course was to design a simple CPU that supported move, move immediate, add, and subtract commands.
